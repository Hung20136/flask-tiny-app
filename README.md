## 2. Mô tả project
Ứng dụng web đơn giản với Flask gồm 2 chức năng chính:
- Blog: Cho phép user tạo, chỉnh sửa, xóa bài viết.

## 3. Hướng dẫn cài đặt & chạy
### Yêu cầu:
- Python 3.x
- Flask
- SQLite (hoặc database khác)
